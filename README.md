# Advanced To-Do List (Tkinter)

A simple yet **beautiful and advanced To-Do List application** built using **Python Tkinter**.
This app helps users manage daily tasks efficiently with features like add, edit, delete, search, and theme switching.

---

## 📂 Project Structure

```
todo-list/
│
├── main.py        # Main application file
├── tasks.txt      # Stores saved tasks
└── README.md      # Project documentation
```

---

## ▶️ How to Run

1. Install Python (if not already installed)
2. Copy the code into a file named `main.py`
3. Run the program:

```
python main.py
```

##  Features

*  Add new tasks
*  Delete selected tasks
*  Edit existing tasks
*  Clear all tasks
* Mark tasks as completed (double-click)
*  Search/filter tasks in real-time
*  Save tasks to a file (persistent storage)
*  Toggle between Dark & Light themes
*  Scrollable task list

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI Library)
* File Handling (for saving tasks)
---

## 💡 How It Works

* Tasks are stored in a file named `tasks.txt`
* Each task is displayed with a checkbox:

  * `[ ]` → Not completed
  * `[✔]` → Completed
* Double-click a task to toggle completion
* Search bar filters tasks instantly

---

## 🎨 UI Highlights

* Modern font: **Segoe UI**
* Clean button layout
* Dark mode by default
* Smooth theme toggle

---

## ⚠️ Notes

* Make sure `tasks.txt` is in the same folder as the script
* If not present, it will be created automatically
* Do not manually edit the file while the app is running
